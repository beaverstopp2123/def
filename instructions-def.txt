Инструкции по пользованию DEF. 
Для начала установите python3 и pip3 если нет в вашей сборке.
После этого добавляем файл def в /usr/local/bin 
и выполняем команду chmod +x def 
и пишите def.

Команда доступна из любой директории.
Чтобы установить библиотеку пишем 
def install и название библиотеки.
Для удаления библиотеки:
def delete название библиотеки.

Например:
def install colorama - для установки colorama
def delete colorama - для удаления colorama. 

И так чтобы проверить установку библиотеки
пишем def browse название библиотеки

например устанавливаем colorama:
def install colorama 
и проверяем установку
def browse colorama 

Если установка прошла успешно мы увидим 
def browse colorama - и путь к нашей библиотеки.
